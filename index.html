Aquí tienes el archivo actualizado:
📥 Descargar Presentación Interactiva

🔧 Corrección aplicada:

El problema:

La fórmula anterior 420 - (percent / 100) * 380 usaba 420 como referencia (el tamaño del SVG) en vez de 400 (el borde inferior real del círculo). Esto hacía que el 100% quedara en Y = 40 en vez de Y = 20, dejando un espacio vacío arriba.

La solución:

Definí constantes claras y una función pctToY():
JavaScript
Copy
const EARTH_TOP = 20;      // 210 - 190 (centro - radio)
const EARTH_BOTTOM = 400;  // 210 + 190 (centro + radio)
const FILLABLE_HEIGHT = 380; // 400 - 20

function pctToY(percent) {
    // 0% = bottom (400), 100% = top (20)
    return EARTH_BOTTOM - (percent / 100) * FILLABLE_HEIGHT;
}
